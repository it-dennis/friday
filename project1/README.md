# Git & GitHub Übungsrepository

Dieses Repository dient zum Üben des Workflows zwischen **VSCode** und **GitHub**.

## Geplante Übungen
- [ ] Dateien über das Terminal mit `git add`, `commit` und `push` hochladen.
- [ ] Änderungen in `main.py` vornehmen und synchronisieren.
- [ ] Nutzen der Quellcodeverwaltung (Source Control) direkt in VSCode.

## Setup
- **Editor:** Visual Studio Code
- **Git:** Installiert via Homebrew
- **Sprache:** Python
